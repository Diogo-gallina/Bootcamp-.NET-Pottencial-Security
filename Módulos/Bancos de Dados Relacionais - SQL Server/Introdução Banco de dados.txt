#Introdução banco de dados

##Tipos de Banco de Dados

- Um banco de dados é uma coleção organizada de informações estruturadas, normalmente armazenadas eletronicamente em um sistema
de um computador

- Banco de dados relacional: O tipo mais usado atualmente, armazenando dados estruturados, sendo organizados em tabelas, 
com linhas e colunas, que se relacionam entre si
EX:
* SQL Server
* Oracle database
* MySQL
* PostgreSQL

- MySQL é um bd gratuíto, mais direcionado a pequenos sistemas sem muitos acessos simultâneos

- Para sistemas criticos, que precisam estar sempre ou quase sempre online a melhor escolha são os bds SQL sever e Oracle que 
apersar de seren bds pagos, são bancos de dados mais parrudos, pois eles tem um suporte muito melhor para caso ocorra erros


##Entendendo uma tabela

- Tabelas são dados estruturados e organizados logicamente em formato de linhas e colunas


##Banco de dados não relacional

- Banco de dados onde os dados não são armazenados em tabela, e sim armazenados de maneira não estruturada ou semi-estruturada.
EX:
MongoDB

- É mais ultilizado para sistemas que precisam de uma transmissão de dados mais ampla, rápida e flexivel 


##Tipos de dados

- Um dado estruturado é um tipo de dado que segue regras rigidas, se você declarou um tipo de dado pra uma coluna, a coluna 
não ira aceitar outro é tipo de dado ou caso você defina que a coluna não pode aceitar valores nulos, ela não aceitará 
campos vazios

- Um dado semi-estruturado é um dado que tem suas regras de sintaxe, porém as regras não são tão rigidas, pois você pode 
alterar ou adicionar dados sem precisar ser obrigatório para todos


##Entendendo o DBMS

- Database Management System é um software ultilizado para acessar e manipular um sistema de banco de dados

- SQL Server Management Studio é o DBMS que iremos ultilizar, que serve para ter uma maneira visual de acessar o banco de dados
